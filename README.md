# Zanshin

*Remaining mind. Lingering awareness. Perfect focus.*

## About Zanshin

Zanshin (残心) is a osu!mania replay rendering - creating videos that capture not just the gameplay, but the complete experience with perfect synchronization and attention to detail.

## Features

- **High-Quality Rendering**: Render osu!mania replays in stunning 1080p/720p video
- **Audio Synchronization**: Perfect audio-video sync for complete immersion
- **Web-Based**: No downloads required - works directly in your browser
- **4K Support**: Specialized for 4-key osu!mania beatmaps
- **Real-time Preview**: See your renders before finalizing

## Quick Start

1. Load your `.osu` beatmap file
2. Load your `.osr` replay file
3. Select the corresponding audio file
4. Preview the render
5. Export your high-quality video

## Technology

Built with modern web technologies:
- React + TypeScript
- WebCodecs API for hardware-accelerated encoding
- Canvas 2D for precise gameplay visualization
- MP4 muxing for professional video output
