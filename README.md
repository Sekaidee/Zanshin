## Zanshin

Zanshin 残心 is a project I'm working on to render osu!mania replays into videos.
The idea is to capture gameplay in a clean and easy way so replays can be shared or archived. It's still in development and I'm gradually improving it.

## Features

- **Real-time preview** so you can see how the replay will look before rendering.
- **Some visual customization** to adjust the style of the render.
- **Web-based**, meaning you can use it directly in your browser.
- **osu! skins support** planned for future updates.

## Quick Start

1. Load your `.osr` replay file
2. Load your `.osu` beatmap file
3. Select the corresponding audio file
4. Preview the render
5. Export your high-quality video
